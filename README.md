#### Activity 4: Fuel Assembly Modeling
- Models a standard $17 \times 17$ PWR-style fuel assembly.
- Includes detailed geometry for fuel pins, guide tubes, instrument tubes, and control rods (AgInCd).
- Implements burnable poisons using Gadolinia ($Gd_{2}O_{3}$).
- Performs eigenvalue calculations ($k_{eff}$) and generates plots for neutron flux and fission rate distributions.

#### Activity 5: Homogenized Mixture Analysis
- Models a homogenized mixture of fuel and moderator in a spherical geometry.
- Analyzes the effect of moderator-to-fuel volume ratios on criticality.

#### Activity 6: Depletion and Burnup
- Focuses on fuel assembly depletion calculations over time.
- Tracks the change in isotopic composition and reactivity ($k_{eff}$) as a function of burnup.
